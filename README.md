# ひとことボード

匿名で「ひとこと」を投稿できるシンプルな掲示板アプリです。

## 機能

- ひとこと投稿（名前は省略可・本文は140文字まで）
- 投稿一覧の表示（新着順）
- いいねボタン
- **自分の投稿だけ削除できる仕組み**
  - 訪問者ごとに匿名トークン（Cookie）を発行し、投稿に紐付けています
  - ログイン不要のまま「本人の投稿のみ削除可」を実現しています

## 技術スタック

| 役割 | 技術 |
|------|------|
| サーバー | Node.js + Express |
| テンプレート | EJS（サーバーサイドレンダリング） |
| データベース | SQLite（better-sqlite3） |

## セキュリティ面の工夫

- EJSの自動エスケープによるXSS対策
- サーバー側での入力バリデーション（必須チェック・文字数制限）
- 削除権限の検証はサーバー側で実施（owner_tokenの一致確認）
- owner_tokenはビューに渡さない（本人フラグのみ渡す）

## ローカルでの動かし方

```bash
npm install
npm start
# → http://localhost:3000
```

## デプロイ

Render（無料プラン）にデプロイしています。

- Build Command: `npm install`
- Start Command: `npm start`

※ 無料プランのためファイルシステムが永続化されず、再起動時に投稿データはリセットされます。
